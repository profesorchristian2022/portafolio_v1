## Descripción del proyecto

Este proyecto consiste en la creación de una página web de tipo **Portafolio Personal**, desarrollada utilizando tecnologías base del desarrollo web:

- HTML
- CSS

El objetivo principal es que los estudiantes comprendan cómo construir la estructura visual de un sitio web separando correctamente:

- la capa de contenido (HTML)
- la capa de estilos y diseño (CSS)

La página incluye distintas secciones utilizadas comúnmente en portafolios profesionales del área informática, tales como:

- Inicio
- Sobre mí
- Habilidades
- Proyectos
- Formación
- Contacto
